⚙️ First-Time Setup

If this is your first time, complete these steps:

📋 Need the complete Google Sheet structure?
Check the GOOGLE_SHEET_STRUCTURE.md file in your project for all the data to copy-paste.
  1. Create a Google Sheet at sheets.google.com
  2. Create tabs: home, studio, contact, careers, faq, privacy, terms
  3. For each tab, create 2 columns: Key (A) and Value (B)
  4. Copy the data from GOOGLE_SHEET_STRUCTURE.md into each tab
  5. Click File → Share → Publish to web
  6. Select Entire document, Format: CSV
  7. Click Publish
  8. Copy the Sheet ID from the URL (between /d/ and /pub)
  9. Get the GID for each tab from the URL (e.g., #gid=0)
  10. Send the Sheet ID and GIDs to your developer to update the code
  11. Share the sheet with your CEO's email with Editor permission

🌍 How It Works

  • CEO in USA → Opens Google Sheet → Edits content → Saves
  • Website → Automatically fetches latest content → Updates instantly
  • No Git commits needed - Changes are live immediately
  • Works from anywhere - USA, India, anywhere with internet
