� Google Sheets CMS Setup

Your CEO can edit content in Google Sheets from anywhere (USA, India, etc.) and changes will appear instantly on the website.

Step 1: Create Google Sheet

  1. Go to sheets.google.com and create a new sheet
  2. Create these tabs: home, studio, contact, careers, faq, privacy, terms
  3. For the home tab, create 2 columns: Key (A) and Value (B)
  4. Add this data to the home tab:
KeyValue
hero.headingHave an Idea Worth Building?
hero.subheadingTell us what you're working on...
hero.footerNoteFrom Idea to Validation...
audiences.eyebrowTARGET AUDIENCES
audiences.titleBuilt for Visionaries...
audiences.subtitleUsing AI to deliver faster

Step 2: Publish the Sheet

  1. Click File → Share → Publish to web
  2. Select Entire document
  3. Format: CSV
  4. Click Publish
  5. Copy the URL (it will look like: https://docs.google.com/spreadsheets/d/SHEET_ID/pub?output=csv)

Step 3: Get Sheet GIDs

  1. Click on each tab in your Google Sheet
  2. Look at the URL - you'll see #gid=0, #gid=123456789, etc.
  3. Note down the GID for each tab:
    • home: gid=0 (usually)
    • studio: gid=??? (check URL)
    • contact: gid=??? (check URL)
    • etc.

Step 4: Send the Developer This Info

Send your developer (you) these two things:

  1. Sheet ID - the long ID in the URL (between /d/ and /pub)
  2. GIDs - the gid number for each tab
How it works:
Once set up, your CEO can edit the Google Sheet from anywhere. The website will automatically fetch the latest content. No Git commits needed!

Step 5: Share with CEO

  1. Click Share button in Google Sheet
  2. Add your CEO's email with Editor permission
  3. They can now edit content from anywhere in the world
