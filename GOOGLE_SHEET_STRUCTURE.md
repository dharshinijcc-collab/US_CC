# Google Sheet Complete Structure

## How to Set Up Your Google Sheet

### Step 1: Create the Sheet
1. Go to sheets.google.com
2. Create a new sheet
3. Create these tabs: `home`, `studio`, `contact`, `careers`, `faq`, `privacy`, `terms`

### Step 2: For Each Tab, Create 2 Columns
- Column A: **Key**
- Column B: **Value**

## Important Notes:

1. **Array items use numbers**: For lists (like benefits, services, etc.), use `.0`, `.1`, `.2` etc.
   - Example: `foundersInvestors.founders.benefits.0` = "First benefit"
   - Example: `foundersInvestors.founders.benefits.1` = "Second benefit"

2. **Nested objects use dots**: Use `.` to separate nested levels
   - Example: `hero.heading` = "Main heading"
   - Example: `services.services.0.title` = "First service title"

3. **Copy exactly**: Copy the Key and Value exactly as shown above

4. **After setup**: 
   - Publish the sheet (File → Share → Publish to web → CSV)
   - Send me the Sheet ID and GIDs for each tab
