# Crestcode Backend - Completely Free Setup

This backend uses 100% free services with NO trials or time limits.

## Free Services Used

1. **Vercel** - Hosting (Free forever, 100GB bandwidth/month)
2. **Supabase** - Database (Free tier: 500MB database, unlimited requests)
3. **Resend** - Email (Free tier: 3,000 emails/month)

## Setup Instructions

### Step 1: Create Supabase Account (Free)
1. Go to https://supabase.com
2. Sign up for free (no credit card required)
3. Create a new project
4. Go to Project Settings > API
5. Copy:
   - Project URL (SUPABASE_URL)
   - anon public key (SUPABASE_ANON_KEY)

### Step 2: Create Database Tables
In Supabase SQL Editor, run:

```sql
-- Idea submissions table
CREATE TABLE idea_submissions (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  idea TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Contact submissions table
CREATE TABLE contact_submissions (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  first_name TEXT NOT NULL,
  work_email TEXT NOT NULL,
  company TEXT,
  service_interest TEXT,
  project_stage TEXT,
  message TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Enable RLS (Row Level Security)
ALTER TABLE idea_submissions ENABLE ROW LEVEL SECURITY;
ALTER TABLE contact_submissions ENABLE ROW LEVEL SECURITY;

-- Allow public insert
CREATE POLICY "Allow public insert on idea_submissions" 
ON idea_submissions FOR INSERT WITH CHECK (true);

CREATE POLICY "Allow public insert on contact_submissions" 
ON contact_submissions FOR INSERT WITH CHECK (true);
```

### Step 3: Create Resend Account (Free)
1. Go to https://resend.com
2. Sign up for free (no credit card required)
3. Go to API Keys
4. Copy your API key (RESEND_API_KEY)
5. Verify your domain or use Resend's default

### Step 4: Install Dependencies
```bash
cd api
npm install
```

### Step 5: Set Environment Variables
1. Copy `.env.example` to `.env`
2. Add your Supabase and Resend credentials

### Step 6: Deploy to Vercel (Free)
```bash
npm install -g vercel
vercel login
vercel
```

Follow the prompts to deploy. Vercel will automatically detect the API routes.

### Step 7: Add Environment Variables in Vercel
1. Go to your Vercel project dashboard
2. Settings > Environment Variables
3. Add:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
   - RESEND_API_KEY

## API Endpoints

- POST `/api/submit-idea` - Submit idea from landing page
- POST `/api/submit-contact` - Submit contact form
